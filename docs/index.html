Tubby1981 System Ansible Collection

Ansible collection voor Linux systeembeheer, met rollen voor **Foomuuri multizone firewall**, **Apache webserver**, **MariaDB database server**, en volledige ondersteuning voor **PHP** en **phpMyAdmin** configuratie.

Quick Install

ansible-galaxy collection install tubby1981.system

Key Features

  • **Foomuuri Firewall:** Multi-zone architectuur, Geavanceerde NAT (SNAT/DNAT/masquerading), Dynamische IP-lijsten van DNS of URL's.
  • **Apache Webserver:** Geautomatiseerde Let's Encrypt SSL/TLS voorziening, beveiligingsverbeteringen, multi-domein ondersteuning, en pre-flight DNS validatie.
  • **MariaDB Database:** Installatie en beveiliging van MariaDB/MySQL server, beheer van gebruikers/databases, configuratie van replicatie (Master/Slave), en finetuning van prestatie-instellingen.
  • **PHP:** Installatie en configuratie voor **PHP-versies 7.4 tot 8.4**.
  • **phpMyAdmin:** installatie en configuratie van **phpMyAdmin** voor eenvoudig databasebeheer.
