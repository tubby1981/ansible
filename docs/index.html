Tubby1981 System Ansible Collection

Ansible collection for Linux system management featuring Foomuuri - a modern multizone bidirectional nftables firewall.

Quick Install

ansible-galaxy collection install tubby1981.system

Features

  • Multi-zone firewall architecture
  • Advanced NAT (SNAT/DNAT/masquerading)
  • Dynamic IP lists from DNS or URLs
  • Zone mapping for advanced routing
  • Traffic marking and QoS support
